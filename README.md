# heropatterns

[tailwindcss](https://tailwindcss.com) plugins for dynamically generating [heropatterns](https//heropatterns.com)

## Usage

```
bgp-[pattern]-[opacity]-[color]
```

for example, `bgp-topography-.5-gray-700 bg-gray-900`
